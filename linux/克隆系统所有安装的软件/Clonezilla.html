              Clonezilla

              Clonezilla 是分区和磁盘镜像克隆程序，类似于 True Image® 或 Norton Ghost®。它可以帮助你做到系统部署、 裸金属备份和恢复。

              Clonezilla 是分区和磁盘镜像克隆程序，类似于 True Image® 或 Norton Ghost®。它可以帮助你做到系统部署、 裸金属备份和恢复。有两种类型的Clonezilla都是可用的，Clonezilla live 和 Clonezilla SE （服务器版）。Clonezilla live 适用于单机备份和还原。虽然 Clonezilla SE 主要用于进行大规模部署，它同时可以克隆很多 （40+！） 计算机。Clonezilla 保存和恢复只使用硬盘块，这提高了克隆效率。据报道，在采用一些高端硬件的 42 节点群集中，多播的恢复速度达到 8 GB/min。

              特点︰
              • 多种文件系统支持: （1) GNU/Linux 的 ext2、 ext3、 ext4、 reiserfs、 reiser4、 xfs、 jfs，btrfs、 f2fs 和 nilfs2 ； （2) MS Windows 的 FAT12、 FAT16、 FAT32、 NTFS 的； （3） Mac OS 的 HFS + 的 ，FreeBSD UFS； （4) NetBSD 和 OpenBSD；（5) minix 的 Minix，和 （6) VMWare ESX 的 VMFS3 和 VMFS5 的 。因此你可以克隆 GNU/Linux，MS windows，基于英特尔处理器的 Mac OS、 FreeBSD、 NetBSD，OpenBSD，Minix，VMWare ESX 和 Chrome OS/Chromium OS，无论是 (x86) 32 位或 64 位 (x86-64) 操作系统。对于这些文件系统，只使用块分区保存和恢复。
                对于不支持的文件系统，Clonezilla 是通过 dd 实现扇区-扇区拷贝 。
              • GNU/Linux 下的 LVM2 （LVM 版本 1 不支持） 是支持的。
              • 引导加载程序，包括 grub （版本 1 和版本 2） 和 syslinux，可能需要重新安装。
              • 支持硬盘的 MBR 和 GPT 分区格式。Clonezilla live 还可以在 BIOS 或 uEFI 的计算机上启动。
              • 支持无人参与的模式。几乎所有的步骤，可以通过命令和选项。您还可以使用大量的引导参数以自定义您自己的镜像和克隆。
              • 支持一个镜像恢复到多个本地设备。
              • 镜像可以被加密。这是通过 ecryptfs，POSIX 兼容的企业加密文件系统。
              • Clonezilla SE 适用于大规模多播克隆。也可以远程使用它来保存或恢复一组计算机，如果在您的客户端支持 PXE 和网络唤醒。
              • 镜像文件可以位于本地磁盘、ssh 服务器、samba 服务器、NFS 服务器或 WebDAV 服务器。
              • 可以用 AES 256 加密来保护数据访问、 存储和传输。
              • 基于 Partclone （默认值）、 Partimage （可选）、 ntfsclone （可选） 或 dd 镜像或克隆的分区。然而，Clonezilla 包含一些其他程序，可以保存和恢复分区，也是整个磁盘。
              • 通过使用另一个免费软件 drbl-winroll，也是由我们开发的，可以自动更改主机名、 组和克隆 MS windows 机器的 SID。

              Clonezilla live 的最低系统要求︰

              • X86-64 处理器
              • 196 MB 的系统内存 (RAM)
              • 启动设备，例如 CD/DVD 驱动器、 USB 数码港、 PXE 或硬盘
              局限性︰
              • 目标分区必须等于或大于源分区。
              • 差异备份尚未实现。
              • 在线镜像克隆尚未实现。要镜像或克隆的分区，必须卸载。
              • 由于镜像格式的限制，镜像不可以浏览或安装。你不可以从镜像中恢复单个文件。然而，你仍然有变通办法实现，读这。
              • Clonezilla live 尚未实现恢复多个 CD 或 DVD。如果你选择创建恢复 iso 文件，现在所有的文件必须在一张 CD 或 DVD。
