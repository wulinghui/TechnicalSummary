# 规则提取
## 对app等其他
对于产品不做限制，对小程序啥的产品。是小程序自身的限制使用。
## 禁止
- 不得利用本服务从事虚假交易。
- 如有二级商户，则当识别并保存二级商户的身份信息和经营资质。（同时防止二级商户违法活动）[简单地说不分级,都算你的名义]
- 不得主动或协助持卡人进行信用卡套现、洗钱、交易分单等违法经营行为，不得出现虚假申请、恶意倒闭等严重风险行为。 【违法行为】
- 不得用同一微信支付商户号同时经营实物类和虚拟类的商品或服务。 【得用2个商户号】
- H5支付只能自己使用。
